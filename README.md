# InfnetFood
Aplicativo desenvolvido em React Native com Expo, simulando um sistema de pedidos e delivery de alimentos.

## Funcionalidades
- Login com autenticação simulada
- Listagem de categorias e produtos
- Carrinho de compras
- Checkout
- Mapa com restaurantes
- Tema claro/escuro
- Notificações
- Consumo de API externa (DummyJSON)

## Como executar
- Instalar dependências: npm install
- Rodar projeto: npx expo start
- Executar no celular via Expo Go
